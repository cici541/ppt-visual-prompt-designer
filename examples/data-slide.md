# Data Slide Example

## Input

```text
不用问，直接生成。

页面内容：2024 年三个渠道收入分别是直销 1200 万、代理 800 万、线上 450 万。
核心表达：直销仍是最重要的收入来源。
```

## Expected Behavior

The skill should:

- Identify the page as `数据页 / Data`
- Choose a bar chart or column chart because the content is a quantity comparison
- Use default output mode: bottom-image mode
- Keep exact labels and numbers editable in PPT

## Example Final Prompt Excerpt

```text
生成一张 16:9 专业商务演示页底图，用于表现 2024 年收入渠道对比。右侧构建三项横向条形图的无文字结构：第一条最长并作为视觉焦点，下面两条较短并使用低饱和中性色。所有标题、图表标签、精确数字、来源备注和正文都应保留为后期 PPT 可编辑元素，因此画面中不要出现任何可读文字、中文、英文、数字、logo、水印或伪造数据标签。
```
